Quem está por trás deste site?

Que ferramentas foram usadas?

Usámos uma grande variedade de ferramentas, tendo quase todas em comum o facto de ser software livre. Também criámos alguns programas e scripts para obter o que queríamos, estando todo esse trabalho disponível no nosso repositório no Gitorious. Segue-se uma lista exaustiva de todos os programas que nos tornaram a vida mais fácil:

  • Python para processar toda a informação para incluir no website, com a benesse de bibliotecas como o feedparser, dateutil, e python-twitter.
  • Django para criar todo o motor do website, puxando toda a informação da base de dados para a apresentar de uma forma linda e útil.
  • Gimp e Inkscape
  • Gedit, vim e .... para editar código e texto
  • Google Refine

Feito durante a maravilhosa Hackathon no Hacklaviva Porto 2010.
